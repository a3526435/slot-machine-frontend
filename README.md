# Slot Machine Frontend

这是一个基于React的前端应用，用于连接和交互一个部署在Blast链上的智能合约老虎机游戏。

## 先决条件

在开始之前，请确保你已经在你的系统上安装了以下软件：

- Node.js (推荐使用LTS版本)
- npm (Node.js自带的包管理工具)
- MetaMask (浏览器扩展，用于管理以太坊钱包)
